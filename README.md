# Audris Li Portfolio

Static portfolio website for Audris Li.

## Preview locally

Serve the repository root with any static file server, for example:

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173`.

## Notes

- Main project metadata is loaded from `data/project_metadata_draft.csv`.
- The Photography sequence is defined in `src/data/photography-sections.js`.
